NeedMorseCowbell
================

For when your life needs morse code

Installation
------------

```bash
git clone https://github.com/mattbornski/NeedMorseCowbell.git
cd NeedMorseCowbell
virtualenv --no-site-packages -p`which python2.7` env
source env/bin/activate
pip install -r requirements.txt
python service.py
```
